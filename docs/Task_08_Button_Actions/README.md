# 📌 Project Overview

This repository contains the **Verify Button Actions** Playwright automation task developed using **JavaScript**.

The goal of this task is to automate different button interactions including **Double Click**, **Right Click**, and **Dynamic Click**, and verify that the appropriate success message is displayed while following industry-standard automation framework practices and Git workflows.

# ▶️ Installation

Clone the repository

```bash
git clone https://github.com/Umer7787/playwright-javascript-automation-framework.git
```

Navigate into the project

```bash
cd playwright-javascript-automation-framework
```

Install dependencies

```bash
npm install
```

Install Playwright browsers

```bash
npx playwright install
```

---

# ▶️ Run Tests

Run the test

```bash
npx playwright test tests/Task_08_Verify_Button_Actions/ButtonActions.spec.js
```

Run in headed mode

```bash
npx playwright test tests/Task_08_Verify_Button_Actions/ButtonActions.spec.js --headed
```

Run in debug mode

```bash
npx playwright test tests/Task_08_Verify_Button_Actions/ButtonActions.spec.js --debug
```

# 🌿 Git Workflow

Every task is developed in its own feature branch.

Example:

```text

├── feature/task-01-login-functionality

├── feature/task-03-page-title
├── feature/task-04-checkbox-validation
```

After review, each feature branch is merged into the `main` branch.
